PROMEDIO I

En este promedio I, aprendimos el uso de visual studio code asi como el uso de las diferencias de index, landing y los comandos para la configuracion de la pagina

Sesión 1

En nuestra primera clase de Laboratorio de Innovación 2, exploramos Visual Studio Code, un esencial editor de código desarrollado por Microsoft para múltiples sistemas operativos y la web. Con su capacidad para escribir, editar y depurar código de forma eficiente, este software ofrece diversas características y extensiones que facilitan el proceso de programación.

Sesión 2

En la segunda sesión, adquirimos conocimientos sobre las definiciones de bitácora y las distinciones entre blog y página web. Posteriormente, procedimos a crear nuestro primer documento HTML denominado índice.

  • Más Información

Sesión 3

Durante la tercera clase, evaluamos nuestro progreso utilizando Google y exploramos las diversas interfaces y lenguajes de programación web, tales como HTML, CSS y JS.

  • Más Información

Sesión 4

En la cuarta sesión, exploramos algunas de las capacidades de Google Sites, incluyendo la creación de una página de inicio, la personalización de diseños y otras funcionalidades. También examinamos minuciosamente el menú de navegación.

  • Más Información

Sesión 5

Durante la quinta sesión, exploramos la creación de una página de inicio, abordamos técnicas de personalización de diseños y profundizamos en el análisis del menú de navegación.

  • Más Información

Sesión 6

En la sexta clase, Hoy aprendimos acerca de nuevos recursos para maximizar el uso del CSS. Estos recursos nos permitirán superponer elementos, crear efectos, añadir movimiento y realizar diversas modificaciones a nuestros textos e imágenes..

  • Más Información

Sesión 7

En la septima clase, Hoy adquirimos conocimiento sobre un código adicional destinado a mejorar la interacción del usuario con nuestra página web. Es esencial para el usuario final que haya claridad en cuanto a las acciones previas realizadas en la misma página, por lo que ajustamos los detalles de la interacción con el puntero. .

  • Learn more

Sesión 8

Hoy exploramos un código adicional para mejorar la interacción en nuestra página web. Nos organizamos en grupos durante la clase, donde el profesor nos presentó dos plataformas: Netlify, que nos permite alojar los archivos de nuestra página web, y GitHub, que está vinculada a Netlify. Estas herramientas nos facilitarán la creación de un enlace personalizado para nuestra página, evitando tener que subir nuestro trabajo como archivos comprimidos y permitiéndonos compartirlo fácilmente como un enlace directo.

  • Learn more

Sesión 9

En la novena sesión exploramos Arduino, que consta de una placa de circuito programable, comúnmente conocida como microcontrolador, junto con un software o entorno de desarrollo integrado (IDE) que se instala en tu computadora. Este software te permite escribir y cargar el código en la placa física.

  • Learn more

Sesión 10

En la décima sesión presencial exploramos Arduino y aprendimos sobre la creación de circuitos. Se destacó el papel del circuito integrado que funciona como el cerebro o procesador de la placa de Arduino, encargado de ejecutar el código de programación.
